# Ecommerce Funnel Analysis

End-to-end product analytics project using SQL and Power BI to analyze customer behavior, identify conversion bottlenecks, measure funnel performance, and generate data-driven business recommendations.

## Dashboard Preview

[Dashboard](dashboard SS.png)

## Business Problem

The company wanted to understand where users drop off during the purchase journey and identify opportunities to improve conversions.

## Tech Stack

- SQL
- Power BI
- Data Analysis
- Funnel Analysis
- KPI Reporting

## Key Metrics

- Total Visitors: 1,000
- Revenue: 324K
- Conversion Rate: 31%
- View-to-Cart Rate: 61.5%
- User-to-Cart Drop-off: 38.5%
- Cart-to-Purchase Drop-off: 49.76%

## Key Insights

- Checkout stage had the highest customer drop-off.
- Mobile users generated the highest revenue.
- Funnel optimization opportunities were identified to improve conversions.
