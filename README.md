# Ecommerce Funnel Analysis

## Project Overview

This project analyzes customer behavior across an e-commerce conversion funnel using SQL and Power BI. The objective was to identify drop-off points in the customer journey, measure conversion performance, and generate actionable recommendations to improve revenue and customer acquisition efficiency.

## Business Problem

The company wanted to understand why a large percentage of visitors were not completing purchases despite healthy website traffic.

Key questions addressed:

* How many visitors progress through each funnel stage?
* Where do customers drop off?
* Which devices generate the highest revenue?
* What opportunities exist to improve conversion rates?

## Key Insights

### 1. Checkout Stage is the Biggest Bottleneck

Nearly 50% of customers who added products to their cart did not complete their purchase.

This suggests friction during checkout, including:

* Complex checkout flow
* Unexpected costs
* Payment issues
* Poor mobile experience

### 2. Mobile Users Drive Revenue

Mobile users generated:

* 226K revenue
* 217 purchases

compared to desktop users, making mobile optimization a high-impact opportunity.

### 3. Overall Funnel Conversion

Out of 1,000 visitors:

* 615 added products to cart
* 309 completed purchases

resulting in an overall conversion rate of 31%.

## Recommendations

### Improve Checkout Experience

* Reduce checkout steps
* Enable guest checkout
* Simplify payment process

### Recover Abandoned Carts

* Email reminders
* Push notifications
* Discount incentives

### Optimize Mobile Experience

* Improve page speed
* Simplify mobile checkout
* Test mobile payment options
